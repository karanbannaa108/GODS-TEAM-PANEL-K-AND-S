# Payment Verification Website

This project contains a simple and professional UTR validation and admin login panel system.

## 📂 Files Included
- **index.html** → Main Page (QR display and UTR validation)
- **login.html** → GODS TEAM PANEL admin login page
- **script.js** → Handles validation and redirection logic
- **1000193709.jpg** → QR code image (placeholder)

## ⚙️ How It Works

### Step 1️⃣: Open `index.html`
- User sees the QR image (1000193709.jpg)
- Enters a 12-digit UTR number
- If valid → Upload Screenshot section unlocks

### Step 2️⃣: Upload Screenshot & Continue
- After verifying UTR, user clicks “Share Screenshot”
- Redirects to the login page (`login.html`)
